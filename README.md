Field phenotyping data analysis from KAUST internship. Includes trait distribution, species comparison (t-test), and automated plot summarization in R.

# [Plant Phenomics & Field Data Analysis](https://github.com/daniil-11-ger/plant-phenomics-kaust)
**Context:** Research internship at KAUST (Jesse Poland Lab).

 **Key Skill:** Advanced R (Tidyverse), Hypothesis Testing, and Phenotypic Visualization (ggplot2).
